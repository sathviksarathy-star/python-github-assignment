# Monthly Savings Estimator

The **Monthly Savings Estimator** is a beginner friendly Python program that helps users estimate how much money they will save in a month based on today's savings. It also compares the estimated monthly savings to a preset monthly goal and provides feedback. This project demonstrates clean code, error handling, and a GitHub-ready workflow.

## How to Run the Program

Make sure Python is installed.  
From the terminal, navigate to the folder containing `app.py` and run:

```bash
python app.py
# or
python3 app.py
